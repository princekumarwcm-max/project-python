1. Tic Tac Toe Game
Developed a responsive Tic Tac Toe game using Python (Tkinter) / HTML, CSS, JavaScript.
Implemented two-player gameplay with automatic win, draw, and reset functionality.
Designed an interactive and user-friendly interface with real-time game updates.

Tech Stack: Python (Tkinter) / HTML, CSS, JavaScript

2. Digital Clock
Built a real-time digital clock displaying current time, date, and AM/PM format.
Used system time functions to update the display every second.
Designed a clean and responsive user interface.

Tech Stack: Python (Tkinter) / HTML, CSS, JavaScript

3. Calculator
Developed a basic calculator supporting arithmetic operations (+, −, ×, ÷).
Implemented input validation and clear/reset functionality.
Created a responsive and easy-to-use interface.

Tech Stack: Python (Tkinter) / HTML, CSS, JavaScript

4. To-Do List
Built a task management application to add, edit, delete, and mark tasks as completed.
Stored tasks locally for persistent data.
Designed a simple and intuitive user interface.

Tech Stack: Python / HTML, CSS, JavaScript

5. Weather App
Developed a weather application that fetches real-time weather data using an API.
Displayed temperature, humidity, wind speed, and weather conditions.
Implemented location-based weather search.

Tech Stack: Python / JavaScript, OpenWeather API
